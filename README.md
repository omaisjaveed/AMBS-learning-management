# AMBS CMS

A modern, scalable, and fully dynamic Content Management System (CMS) built with **React.js**, **Node.js**, **Express.js**, and **SQL**. The platform is designed to provide complete administrative control, allowing administrators to manage website content, users, and features through an intuitive admin dashboard.

---

## 📂 Repository Structure

```text
ambs-cms/
├── frontend/      # React.js Frontend
├── backend/       # Node.js & Express.js Backend
└── README.md
```

---

## 🚀 Features

* Fully dynamic and admin-controlled CMS
* Comprehensive admin dashboard
* Website page and content management
* User management system
* Role-based access control (RBAC)
* Media and content handling
* Analytics and dashboard insights
* Secure authentication and authorization
* RESTful API architecture
* SQL database integration
* Scalable and modular project structure

## ⚙️ Installation & Setup

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/ambs-cms.git
cd ambs-cms
```

### 2. Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

### 3. Backend Setup

```bash
cd backend
npm install
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file inside the `backend` directory and add the following configuration:

```env
PORT=5000

DB_HOST=localhost
DB_PORT=3306
DB_NAME=your_database
DB_USER=your_username
DB_PASSWORD=your_password

JWT_SECRET=your_secret_key
```
