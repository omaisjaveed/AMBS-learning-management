AMBS CMS

A modern full-stack Content Management System (CMS) built with React.js, Node.js, Express.js, and SQL. This system provides a fully admin-controlled platform where all website content and features are managed through the admin panel.

📂 Repository Structure
ambs-cms/
├── frontend/     # React Frontend
├── backend/      # Node.js & Express Backend
└── README.md
🚀 Features
Fully dynamic CMS (admin-controlled system)
Complete admin panel access for all content
Page/content management
User management system
Role-based access control
Media/content handling
Dashboard with analytics overview
RESTful API architecture
Secure authentication system
SQL database integration
Scalable and modular structure

⚙️ Installation
1. Clone Repository
git clone https://github.com/your-username/ambs-cms.git
cd ambs-cms
2. Frontend Setup
cd frontend
npm install
npm run dev
3. Backend Setup
cd backend
npm install
npm run dev
🔑 Environment Variables

Create a .env file inside backend:

PORT=5000

DB_HOST=localhost
DB_PORT=3306
DB_NAME=your_database
DB_USER=your_username
DB_PASSWORD=your_password

JWT_SECRET=your_secret_key
